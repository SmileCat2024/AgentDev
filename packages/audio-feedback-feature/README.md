# @agentdev/audio-feedback-feature

Audio feedback feature for AgentDev - plays sound notifications on call completion

## Installation

```bash
npm install @agentdev/audio-feedback-feature
```

## Usage

```typescript
import { AudioFeedback } from '@agentdev/audio-feedback-feature';

const agent = new Agent({ ... }).use(new AudioFeedback());
```
